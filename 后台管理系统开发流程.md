# 后台管理系统开发流程

## 一，创建项目

### 1,项目git托管，上传githup

```
git inin
git status
git add .
git commit -m '初始化'
git remote add origin git@github.com:jiabei6666/demo.git
git push -u origin master
```



### 2，安装element-框架

```
npm i element-ui -S
```

### 3，创建+切换分支（登录页面）

```
git checkout -b dev-longin
注：每完成一个小功能commit
    push操作master去完成
```

### 4，完成login页面组件，配置相关路由，git commit 提交

### 5，封装axios网路请求模块
